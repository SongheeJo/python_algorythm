```python
# [문제 출처] 백준, https://www.acmicpc.net/problem/11718


# <idea>
# 입력받은 그대로 출력하기
# A = input
# print(A)
# 100줄 이하 출력하기
# try~except

while True : # 무한루프
    try :
        i = 0
        if i == 100:               # 입력은 최대 100줄
            pass
        else:
            A = input()
            0 < len(A) <= 100      # 각 줄은 100글자를 넘지 않으며, 빈줄은 주어지지 않는다.
            print(A)
    except:
        break
        
        
# 입력 설명 중 "알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다. "와 "각 줄은 공백으로 시작하지 않고, 공백으로 끝나지 않는다." 부분이 구현해야되는 조건이라고 이해했는데,
# 이것은 그런 입력이 주어진다는 것이지, 이런 입력이 주어질 때 출력하지 말아야 한다는 뜻이 아니라서 이 경우는 아예 생각할 필요가 없었다.
# 내일은 생각할 필요가 없는 내용은 빼고, EOFError를 이용하여 간단하게 풀어보자.
# 또 하루는 sys.stdin.readline()을 이용해서 풀자.
```
